# Citation Format

Brief citations live inline in the brief. Full citations live in a "Sources" section at the end of the brief.

## Inline Format

Use a footnote-style superscript-like marker, but with a numeric label inside square brackets, so plaintext editors stay readable:

```
BBVA trained 250 senior leaders before anyone else got access [1].
Adoption hit 83% weekly use across the company [1].
Lumen's sellers save about 4 hours a week [2].
```

## Sources Block

At the bottom of the brief:

```
## Sources

[1] BBVA, "Inside BBVA's GenAI rollout: 250 leaders first."
    BBVA.com, 2026-02-14. https://example.com/...
[2] Reuters, "Lumen Copilot Olympics save $50M a year."
    Reuters, 2026-01-22. https://example.com/...
```

Each entry has the outlet, the title (in quotes), the publication date in ISO format, and the URL.

## Direct-Quote Format

When a claim is a verbatim quote, mark it with quotation marks inline and include the speaker in the citation:

```
"Training leadership first was the key to adoption across the whole company." [1, BBVA CTO quoted]
```

## Internal Sources

If a claim comes from a non-public document (a partner case study you have but the public does not), cite the document name and date, and note "[internal]" so the script writer can decide whether to keep, paraphrase, or omit.

## Open Questions Section

At the bottom of the brief, before Sources, include:

```
## Open Questions

- Two sources disagree on the seat count: 200 vs 250. Need confirmation.
- The 3hr/week figure is from a press release; no independent verification yet.
- Director-level testimony available but uncleared for public attribution.
```

This is where uncertainty lives so the script writer can route around it.
