# Brand Vault

Brand identity and voice rules for this workspace. Stages load specific sections as directed by their own CONTEXT.md.

## Files

| File | Key Sections | Load When |
|------|-------------|-----------|
| `voice-rules.md` | "Hard Constraints" through "What the Voice Is NOT" | Writing any script copy |
| `voice-rules.md` | "Strategic Rationale" | Understanding why voice choices were made (rarely needed) |
| `identity.md` | "One-Sentence Brand", "Audience" | Knowing who the script is for |
| `identity.md` | "Positioning", "Content Mission" | Evaluating whether a script serves the brand |

Voice rules and identity are populated by `setup/questionnaire.md`. Edit them directly if your voice evolves. Never let downstream stages overwrite this folder.
